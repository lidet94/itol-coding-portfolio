**Workout Planner**

This is a simple React application that helps users organise their exercises based on intensity levels.
Users can enter an exercise name, choose a type (Cardio, Strength, Flexibility), select one or more categories (Legs, Arms, Core, Full Body), and assign an intensity level.
Exercises are automatically sorted into Low, Medium, and High intensity columns for easy viewing and management.
